Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript.
Nesse caso o comando foi envelopado com a tag "<code>", é um tipo de fonte monoespaçada, facilida em leitura de códigos, por exemplo, por isso o nome code.

Código-fonte + Pré-formatação

num=int(input('Digite um número: '))
if num % 2 == 0:
    print('O número {num} é par')
else:
    print('O número {num} é ímpar')

Citações completas

Segundo Jeff Noble, em seu livro HTML para Leigos:

A diferença entre elementos inline e um bloco de texto é importante. Os elementos HTML neste capítulo descrevem os blocos de texto

Tag <blocknote> para a formatação espaçada, característica de citações completas.
Tag <em> para deixar em itálico.
Tag <q> para as aspas de citação.
